# 长征攻心战：宣传策略选择模拟

这是大学《中国近现代史纲要》社会实践成果展示网页小游戏。

主题：长征中的攻心战——红军标语、口号、漫画的对敌宣传研究。

## 运行方式

直接双击 `index.html` 即可本地运行。

## 云端 AI 配置

如果部署到 Vercel，请在项目环境变量中添加：

```text
DEEPSEEK_API_KEY=你的 DeepSeek API Key
```

前端会请求 `/api/judge`，API Key 只保存在云端环境变量里，不会暴露在浏览器源码中。

## 文件说明

- `index.html`：网页结构
- `style.css`：页面样式
- `script.js`：游戏交互、规则判断和 AI 分析接口逻辑
- `api/judge.js`：Vercel 云端 AI 代理接口

## 展示说明

游戏通过五步选择模拟红军宣传策略部署，并生成宣传评估战报，用于说明长征时期红军宣传的因地制宜、因敌制宜和多模态传播特点。
